Unenrollment Docs

Unenrollment refers to the process of removing enterprise or administrative control from a managed device. This typically involves bypassing or disabling enrollment mechanisms enforced by system firmware or operating system policies.

$ check_enrollment_status ENROLLED: TRUE $ initiate_unenroll --force WARNING: modifying system protections $ status ENROLLED: FALSE

These documents explore how enrollment systems function, along with technical methods, limitations, and risks involved in interacting with these protections.
